Professional project

MWC 2019

Every year Ericsson has a huge presence at Mobile World Congress, the biggest telecom/tech conference in europe.

For MWC 2019 my team did another take on showcasing mixed reality in an urbanisation scenario, really pushing it to the bleeding technological edge whilst still keeping an intact industry focus. 5G is said to be the first ‘G’ which focuses directly on industry scenarios, and urbanisation is going to be a self explanatory application area.

We developed a two-part application, one running on the iPad and one running on the Oculus Go. The iPad application made the 2D map come to life, and could visualize different information layers on top of the existing 2D map.

It could also simulate wind, cars and pedestrians. On top of this a photo-realistic miniature model was displayed, showcasing how developers and architects can visualize models in real time with a visual fidelity close to that of classical rendering arch-viz techniques.

The second application running on the Oculus Go teleported the visitor to the building site in Sweden, overlaying the planned building directly on top of the real world view.

This showcased how it would look if the visitor instead would be there IRL, with an iPad or a pair of AR-visors being able to overlay the virtual building instead.

I was responsible for modeling all the required assets, shading and I did some scripting as well. It was a very organic project that grew during the development.

It was very well recieved and more is to come.
